# Images

## Photo de profil

Placez votre photo de profil ici avec le nom : `thibaut-profile.jpg`

### Instructions :

1. Renommez votre photo en `thibaut-profile.jpg`
2. Placez-la dans ce dossier (`public/images/`)
3. Formats acceptés : JPG, PNG, WebP
4. Taille recommandée : minimum 400x400 pixels (carrée de préférence)
5. L'image sera automatiquement affichée dans :
   - La page "À propos" (section Fondateur)
   - La page "Formateur Fullstack"

### Notes :

- Si la photo n'est pas trouvée, une icône de remplacement sera affichée
- L'image sera affichée dans un cercle avec une bordure
- L'image s'adapte automatiquement au dark mode
